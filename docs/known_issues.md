# Known Issues

- `note_based_stereo` is experimental. Some stereo or volume movement can sound
  less smooth when avoidance logic has to move emitters.
- Large builds can take a long time to place in Minecraft.
- The default build process teleports the configured build player between
  windows. Do not change dimensions or interrupt the build while it is running.
- Back up your world before running generated functions.
- Minecraft Java Edition 1.16.5 is the current tested target. Other versions
  may need command or block state adjustments.
- No open-source license has been selected yet.
