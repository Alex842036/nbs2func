# 更新日志

[English](CHANGELOG.md) | [简体中文](CHANGELOG.zh-CN.md)

## 0.1.1

### 新增

- 完成七步向导 GUI 的简体中文本地化。
- 增加持久化的英文/简体中文语言选择；首次启动时检测系统语言，并支持英文
  回退。
- 建立完整的中英文公开文档，包括 README、更新日志、GUI 指南、生成模式、
  架构、已知问题和示例指南。
- 增加文档本地化测试，覆盖成对文件、语言链接、版本一致性、中文 README
  链接及过期措辞。

### 修复

- 切换语言时保留当前页面、步骤解锁范围、Generate 解锁状态、配置、歌曲
  摘要、配置档选择、输出名称、用户修改标志、配置路径、生成结果、事件和
  输出日志。
- 切换语言前应用当前页面的有效草稿；草稿无效时取消切换，不销毁控件或
  丢失输入。
- Layout 和 Output 的完成检查改用持久化的 `WizardState` 配置，不再依赖
  重建后隐藏步骤中尚未初始化的 Tk 变量。
- 切换语言后，Generate 页面可恢复成功或失败状态、进度、本地化日志、结果
  以及输出文件夹按钮。
- Summary 对布局、方向、输出格式、datapack 构建方式、速度模式和速度后端
  使用本地化显示，同时保持内部 enum 值不变。
- 已完成步骤使用与语言无关的勾号，不再硬编码英文 `OK` 标记。

## 0.1.0-gui-preview

### 新增

- 带可点击步骤导航的 Tkinter 七步向导 GUI。
- 配置保存/加载与共享 `Nbs2FuncConfig` 往返转换。
- 结构化的生成阶段、进度、提示、警告、输出和错误事件。
- Datapack、原理图及两者组合的输出格式。
- 直接相连的简单函数链构建方式。
- 面向区块加载的大型构建玩家传送分段方式。
- 可选的启动模块和矿车播放辅助。
- 速度控制的 `none`、`report` 和 `command` 模式。
- 1.14.4、1.16.5、1.18.2、1.20.1 和 1.21.1 的精确 Minecraft 配置。
- 支持 Unicode 的 datapack 文件夹名和原理图文件名。
- Windows GUI 与依赖安装启动脚本。

### 变更

- CLI 与 GUI 现在共享 `generate_from_config()` 和同一套配置系统。
- Datapack 与原理图写入器使用共享的结构化构建计划。
- 组合输出会把完整方块结构写入 `.schem`，把仅运行时逻辑写入 datapack。
- 简单函数链输出会把大型构建拆分为直接相连的函数文件，每个文件最多
  65535 条命令。
- GUI 生成日志使用简洁的结构化事件，CLI 诊断仍保持详细。

### 修复

- GUI 生成解锁、完成竞态和导航状态问题。
- 当前阶段进度残留以及空的重试阶段显示。
- 重新生成后残留的旧 datapack 构建文件。
- Unicode 输出名称回退为通用 ASCII 名称的问题。
- 加载配置和重置操作时 GUI 专用状态泄漏的问题。
- 已禁用模块字段中的无效值阻止导航的问题。
- GUI 缺少 datapack 覆盖确认的问题。
- 后台生成期间缺少关闭警告的问题。

### 已知预览限制

- 逐音符立体声布局采用启发式算法，处理大型歌曲时开销可能很高。
- 无法安全取消生成；强制退出可能留下不完整文件。
- 简单函数链输出假定目标区域已经加载。
- 玩家传送输出要求已配置的构建玩家可用。
- GUI 测试和打开输出文件夹功能目前以 Windows 为主。
- 原理图输出不会把召唤实体作为存活实体嵌入。
- 尚未包含手动排列和交互式布局可视化。

完整列表见 [docs/zh-CN/known_issues.md](docs/zh-CN/known_issues.md)。
